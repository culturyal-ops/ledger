## Frequently Asked Questions

### Q: Can we use this for multiple branches?
**A:** Each USB works independently. For Wayanad, you can purchase an additional USB (₹5,000) with the same software. Data is separate per branch. Multi-branch sync is not available in current version.

### Q: What if we need custom features?
**A:** Base package includes all standard features. Custom development (custom reports, purity calculations, integrations) is quoted separately starting at ₹15,000 per feature.

### Q: What happens after 1 year of support?
**A:** Software continues to work forever (lifetime license). Extended support is optional: ₹15,000/year for continued WhatsApp/phone support and updates.

### Q: Can we try before buying?
**A:** Yes. ₹10,000 refundable deposit gets you a 15-day trial with full features. If satisfied, pay remaining ₹65,000. If not, full refund (conditions apply).

### Q: What if USB drive fails?
**A:** You have 2 USBs (primary + backup). If both fail, we can restore from your encrypted backup files. Replacement USB: ₹3,000.

### Q: What if we forget the master password?
**A:** Master password cannot be recovered (by design for security). This is why we provide backup USB and recommend regular encrypted backups to separate location.

### Q: Do you provide on-site installation?
**A:** Not included in base package. Available as add-on: ₹15,000 (Palai) + ₹10,000 (Wayanad). Includes physical setup, in-person training, and data migration assistance.

### Q: Can we export data later?
**A:** Yes. All reports can be printed or saved as HTML. Future versions will include CSV export. You own your data completely.

### Q: What about GST compliance?
**A:** GoldVault tracks transactions and provides audit reports. For GST filing, you'll still use your existing GST software. GoldVault is for internal smith ledger management, not GST filing.
